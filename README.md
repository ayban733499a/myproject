# HTML Project (Landing Page)

## Run
Just open `index.html` in your browser.

## Files
- `index.html` — page content
- `style.css` — styling + new color palette (Indigo/Blue)
- `script.js` — mobile menu, theme toggle, animated stats, form validation

## Customize
- Change the stats numbers via `data-count` in `index.html`.
- Change colors in `style.css` under `:root` and `body.light`.
